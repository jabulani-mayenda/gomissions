Firebase Setup Guide

Follow these steps one time to connect your gallery. It's free and takes about 5 minutes.

1
Go to firebase.google.com

Sign in with your Google account and click "Go to Console".

2
Create a new project

Click "Add project", name it go-missions-international, and click through the steps (you can disable Analytics).

3
Enable Firestore Database

In the left menu → Build → Firestore Database → click Create database → choose Start in test mode → select any location → Done.

4
Enable Storage

In the left menu → Build → Storage → click Get started → choose Start in test mode → Done.

5
Get your config keys

Click the ⚙️ gear icon → Project settings → scroll down to "Your apps" → click the </> Web icon → register the app → copy the firebaseConfig object.

6
Paste the keys into firebase-config.js

Open the file firebase-config.js in your project folder and replace PASTE_YOUR_API_KEY_HERE and the other values with what you copied. Save the file and you're done!
